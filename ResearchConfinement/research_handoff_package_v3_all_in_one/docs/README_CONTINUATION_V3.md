# README CONTINUATION V3

This package consolidates the available research handoff materials into one zip.

Contents:
- `original_inputs/`: prior handoff archives and input markdown
- `engine_inputs/`: engine repo archives used as supporting context
- `framework files/`: equation/framework Python files that were uploaded and used as visible scaffolding
- `continuation_runs/`: accessible run folders and artifacts still present on disk
- `docs/`: continuation prompt, manifest, experiment archive, and math stack summary

Important honesty note:
Some earlier generated transient artifacts described in chat were no longer present on disk when this all-in-one archive was built, so they could not be included byte-for-byte. Those runs are described in the experiment archive.
